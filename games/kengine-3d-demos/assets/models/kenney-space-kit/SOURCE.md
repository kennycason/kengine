# Kenney Space Kit Sample Model

This folder contains a small sample model from Kenney's Space Kit:

- Source: https://kenney.nl/assets/space-kit
- Pack: Space Kit
- Creator: Kenney
- License: Creative Commons Zero, CC0
- Included files: `craft_racer.obj`, `craft_racer.mtl`, `turret_double.obj`, `turret_double.mtl`, `meteor_detailed.obj`, `meteor_detailed.mtl`, `License.txt`

Only three models were vendored from the full asset pack so the demo stays small while still exercising real OBJ/MTL loading with simple meshes, more complex meshes, and a separate UV texture-sampling sample.
