# Kenney Space Kit Sample Texture

This folder contains a small sample image from Kenney's Space Kit:

- Source: https://kenney.nl/assets/space-kit

- Creator: Kenney
- License: Creative Commons Zero, CC0
- Included file: `meteor_detailed.png`

This image is used to exercise file-backed GPU texture loading in the 3D demo. It is a texture-loading sample, not a final authored UV skin for the OBJ model.
